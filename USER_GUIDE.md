## 1. 처음 설치하기 — Windows

### 1단계 — 설치 파일 다운로드

1. 배포 담당자에게 받은 GitHub Releases 링크(또는 USB)에서  
   `haksenbu-analyzer-setup-x.x.x.exe` 파일을 다운로드합니다.
2. 브라우저가 "이 파일은 자주 다운로드되지 않습니다" 경고를 표시할 수 있습니다.  
   → **[유지]** 또는 **[더 보기 → 유지]** 를 클릭하세요.

> 📸 *(스크린샷 자리 — 브라우저 경고 화면)*

---

### 2단계 — 설치 파일 실행

1. 다운로드된 `.exe` 파일을 **더블클릭**합니다.
2. **"Windows가 PC를 보호했습니다"** 라는 SmartScreen 경고창이 뜹니다.

> 📸 *(스크린샷 자리 — SmartScreen 경고 화면)*

3. 창 하단의 **[추가 정보]** 를 클릭합니다.
4. **[실행]** 버튼이 나타나면 클릭합니다.

> 📸 *(스크린샷 자리 — "추가 정보 → 실행" 화면)*

---

### 3단계 — 설치 마법사

1. 설치 언어 선택: **한국어** 선택 → **[확인]**
2. 설치 경로 확인 (기본값 권장) → **[설치]**
3. 설치가 완료되면 **[마침]**
4. 바탕화면 또는 시작 메뉴의 **학생부 문장 분석기** 아이콘으로 실행합니다.

---

## 2. 처음 설치하기 — macOS

### 1단계 — DMG 파일 다운로드

1. `haksenbu-analyzer-x.x.x-universal.dmg` 파일을 다운로드합니다.
2. 다운로드된 파일을 **더블클릭**해 디스크 이미지를 마운트합니다.

> 📸 *(스크린샷 자리 — DMG 창)*

---

### 2단계 — 앱 설치

1. 열린 창에서 **학생부 문장 분석기** 아이콘을 **Applications 폴더**로 드래그합니다.

> 📸 *(스크린샷 자리 — 드래그 화면)*

2. 드래그 후 창을 닫고 Finder의 **응용 프로그램** 폴더에서 앱을 실행합니다.

---

### 3단계 — Gatekeeper 경고 해제 (처음 한 번만)

1. 앱을 처음 실행하면 **"개발자를 확인할 수 없습니다"** 경고가 뜹니다.

> 📸 *(스크린샷 자리 — Gatekeeper 경고 화면)*

2. **[확인]** 을 눌러 경고창을 닫습니다.
3. **시스템 설정** → **개인정보 보호 및 보안** 을 엽니다.
4. 하단에 **"학생부 문장 분석기"는 차단되었습니다** 메시지 옆의  
   **[확인 없이 열기]** 를 클릭합니다.

> 📸 *(스크린샷 자리 — 시스템 설정 보안 화면)*

5. macOS가 비밀번호를 요청하면 입력한 뒤 앱을 다시 실행합니다.

> **이후부터는 경고 없이 바로 실행됩니다.**

## 3. API 키 발급 및 연결

앱이 분석을 수행하려면 **OpenRouter API 키**가 필요합니다.  
키는 무료로 발급되며, 사용량만큼 소액이 차감됩니다.

### 키 발급

1. [https://openrouter.ai/keys](https://openrouter.ai/keys) 에 접속합니다.
2. 구글 계정 또는 이메일로 가입합니다.
3. **[Create Key]** 를 클릭하고 이름을 입력합니다.
4. 생성된 키(`sk-or-...` 형식)를 복사합니다.

> ⚠️ 키는 생성 시 한 번만 표시됩니다. 반드시 안전한 곳에 저장하세요.

### 키 연결

1. 앱 우측 상단의 **동전 아이콘 (💰)** 을 클릭합니다.
2. **API 키 입력** 칸에 복사한 키를 붙여넣습니다.
3. **[저장 및 연결 테스트]** 를 클릭해 ✅ 표시가 뜨면 완료입니다.

> 📸 *(스크린샷 자리 — 설정 모달 화면)*

## 4. 앱 기본 사용법

1. **학년** 과 **활동 항목** 을 선택합니다.
2. **진로 희망** 과 **희망 학과** 를 입력합니다. (선택 사항)
3. **학생부 문장** 칸에 분석할 문장을 붙여넣습니다.
4. **[분석 시작]** 버튼을 클릭합니다. (약 15~30초 소요)
5. 결과 탭을 확인합니다:
   - **핵심 분석** — 종합 점수, DNA 충족도, 구조 흐름
   - **심화 진단** — 상위권 체크리스트, 진학 로드맵, 강점·약점
   - **평가 시뮬레이션** — 3인의 입학사정관 시뮬레이션 평가

### 인쇄 보고서

- 우측 상단 **🖨️ 인쇄 보고서** 버튼 → 포함할 섹션 선택 → A4 미리보기  
- 브라우저 인쇄창에서 **"배경 그래픽 인쇄"** 옵션을 켜면 색상이 정확하게 출력됩니다.

## 5. 자동 업데이트

- 앱이 실행되면 **백그라운드에서 자동으로 새 버전을 확인**합니다.
- 새 버전이 감지되면 헤더에 **"새 버전 감지됨"** 배지가 표시됩니다.
- 다운로드가 완료되면 팝업이 뜹니다:
  - **[지금 재시작]** → 즉시 업데이트 적용
  - **[나중에]** → 다음 앱 종료 시 자동 적용

별도 작업 없이 항상 최신 버전이 유지됩니다.

## 6. 문제 해결

### 앱이 안 열려요

| 증상 | 해결 방법 |
|------|-----------|
| Windows SmartScreen 차단 | [추가 정보] → [실행] 클릭 (2단계 참고) |
| macOS "개발자를 확인할 수 없습니다" | 시스템 설정 → 보안 → [확인 없이 열기] (3단계 참고) |
| macOS에서 아무 반응이 없음 | Finder → 응용 프로그램 → 앱 우클릭 → [열기] 선택 |

### 분석이 실패해요

| 증상 | 해결 방법 |
|------|-----------|
| "API 키가 설정되지 않았습니다" | 동전 아이콘 → API 키 입력 (3단계 참고) |
| "연결 테스트 실패" | 인터넷 연결 확인, 방화벽 설정 확인 |
| 분석 중 오류 발생 | OpenRouter 잔액 확인 ([openrouter.ai/credits](https://openrouter.ai/credits)) |
| 결과가 이상하게 나와요 | 문장을 좀 더 구체적으로 작성 후 재시도 |

### 업데이트가 안 돼요

자동 업데이트가 작동하지 않는 경우 GitHub Releases에서 최신 버전을 직접 다운로드하세요.  
→ 배포 담당자에게 최신 링크를 문의하거나, GitHub 저장소 Releases 탭을 확인하세요.

## 7. 왜 보안 경고가 뜨나요?

Windows SmartScreen과 macOS Gatekeeper는 **코드 사이닝 인증서**가 없는 앱을 기본적으로 차단합니다.

코드 사이닝 인증서는 앱 배포사가 "정식 회사임"을 증명하는 디지털 서명으로,  
연간 수십만 원 이상의 비용이 발생합니다.

이 앱은 **개인/학교 단위 무료 배포** 목적으로 제작되었기 때문에 인증서를 별도로 구매하지 않았습니다.  
앱 자체에는 악성 코드가 없으며, 위의 안내에 따라 경고를 우회하면 정상적으로 사용할 수 있습니다.

소스 코드는 GitHub에 공개되어 있어 누구든 직접 확인할 수 있습니다.
